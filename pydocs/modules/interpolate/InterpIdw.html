Sets the type of nodal function as well as options for computing nodal functions.

Parameters:
  • nodal_func_type (string, optional) – The nodal function methodology: constant (0), gradient plane (1), quadratic (2).

set_search_opts(self: xmsinterp.interpolate.InterpIdw, number_nearest_points: int, use_quadrant_octant_search: bool) → None¶

Sets the search options for how to find the nearest points to the interpolation point. The number of nearest points can be specified as well as whether to find the nearest points in each quadrant or octant.

Parameters:
  • number_nearest_points (int) – the number of nearest points to the interpolation point. These points are used to do the interpolation.
  • use_quad_oct_search (bool) – specifies if the search criterion should find the nearest points in each quadrant (2d) or octant (3d)
