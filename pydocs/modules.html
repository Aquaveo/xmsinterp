Modules¶

Last Update: December 5, 2018

These are the modules currently exposed to python from the xmsinterp library

Geometry¶

The geometry module of the xmsinterp python library contains classes and functions for finding triangles.

  • TriSearch

Triangulate¶

The triangulate module of the xmsinterp python library contains classes and functions for creating and triangulating a TIN from a set of points.

  • Tin

Interpolate¶

The interpolate module of the xmsinterp python library contains classes and functions for performing the following types of interpolation given a set of points: IDW, linear, natural neighbor, and anisotropic.

  • InterpAnisotropic
  • InterpIdw
  • InterpLinear
